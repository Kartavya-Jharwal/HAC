## üöÄ Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Kartavya-Jharwal/HAC.git
   ```

2. Open any project's `index.html` in your browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve
   ```

3. Access projects at:
   - Main: `http://localhost:8000/`
   - ctOS: `http://localhost:8000/ctOS/`
   - SAM: `http://localhost:8000/SAM/`
